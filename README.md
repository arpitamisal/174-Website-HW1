# 🎬 Filming Locations Landing Page

A clean and modern **frontend webpage** showcasing filming locations, built using **HTML and CSS**. The page includes a styled hero section, call-to-action, newsletter signup, and footer with multiple office locations.

---

## 🚀 Features

- 🎯 Hero section with “Location Favourites” highlight
- 🎨 Styled "Say Hello" section with background typography
- 📩 Newsletter signup form
- 🔗 Navigation using anchor links across pages
- 📍 Multi-location footer (Edinburgh, New York, London, Manchester)
- 💻 Clean and responsive layout using Flexbox & Grid

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Responsive Design Principles
